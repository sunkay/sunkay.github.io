STEP 1: Install XCode and SDK
-----------------------------
XCode should be installed on your Mac. 

To test if the SDK is installed run the folllowing command:
  - meteor install-sdk ios

STEP 2: Add IOS Platform
------------------------
meteor add-platform ios
